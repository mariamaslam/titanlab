# Titan Multimodal Lab 🚀

Titan Multimodal Lab is a high-performance AI workbench designed to leverage the full power of Google's **Gemini 3.1** models. Built with a futuristic, cyberpunk-inspired interface, it provides a seamless environment for real-time multimodal inference across text, vision, audio, and video.

## ✨ Key Features

- **🔴 Multimodal Neural Engine**: Switch seamlessly between Text, Vision, Audio, and Video modalities.
- **👁️ Spectral Vision**: Upload images for deep structural and contextual analysis.
- **🎙️ Acoustic Intelligence**: Process audio files with live waveform simulation.
- **🎥 Temporal Video AI**: Analyze video frames using advanced temporal reasoning.
- **⚡ TPU-Simulated UI**: Interactive "Titan Core" dashboard featuring real-time system logs, hardware load simulation, and latency tracking.
- **🔐 Secure Kernel**: Built-in validation and error handling for robust API interactions.
- **🎨 Glass-Morphic Design**: A polished technical UI built with Tailwind CSS v4 and Motion.

## 🛠️ Tech Stack

- **Frontend**: React 19, Vite, TypeScript
- **Styling**: Tailwind CSS v4 (Modern Engine)
- **Animations**: Motion (Framer Motion)
- **AI Core**: Google Gemini Generative AI SDK (`@google/genai`)
- **Icons**: Lucide React

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- A Google Gemini API Key (get it at [Google AI Studio](https://aistudio.google.com/))

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/titan-multimodal-lab.git
   cd titan-multimodal-lab
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment Variables:**
   Create a `.env` file in the root directory and add your API key:
   ```env
   VITE_GEMINI_API_KEY=YOUR_API_KEY_HERE
   ```

4. **Start the Development Server:**
   ```bash
   npm run dev
   ```

## 📖 Usage

1. **Select Modality**: Use the sidebar to choose between Text, Vision, Audio, or Video modes.
2. **Upload Context**: Click the upload icon to attach files relevant to your selected modality.
3. **Analyze**: Enter your query in the terminal and hit Enter to trigger the "Titan Core" inference.
4. **Monitor Logs**: Watch the system logs at the bottom for real-time processing updates and latency metrics.
